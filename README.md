# Tempo Contracts

10 Solidity smart contracts deployed on Tempo mainnet — beginner friendly.

## What is Tempo?
Tempo is a payments-focused L1 blockchain backed by Stripe & Paradigm.
- EVM compatible
- Sub-second finality
- Gas paid in USDC — no native token needed
- Fees under $0.001 per transaction

## Contracts

## How to Deploy
1. Open [Remix IDE](https://remix.ethereum.org)
2. Copy any contract from the `contracts/` folder
3. Compile and deploy to Tempo mainnet
4. Confirm in MetaMask
